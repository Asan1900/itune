# iTuneNote

A modern guitar tuner and tablature editor built with React, TypeScript, and Vite.

## Features

- **Guitar Tuner**: Real-time pitch detection with visual feedback
- **Tab Editor**: Create and edit guitar tablature with undo/redo support
- **Tab Library**: Organize and manage your tabs
- **Multi-language Support**: Available in multiple languages
- **Audio Settings**: Customize microphone input and sensitivity

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
